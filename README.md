# 閲覧 URL（最初にここへアクセス）

この GitHub Pages は以下の URL で閲覧できます。  
**https://22-soeda.github.io/share_html_pages/**

## このリポジトリについて

このリポジトリは、静的 HTML/CSS ページを GitHub Pages で公開するためのものです。  
ルートにある `index.html` がトップページとして配信され、`styles.css` で見た目を調整しています。

## 使い方

1. ページを見る  
   上記の URL にアクセスしてください。
2. 内容を編集する  
   `index.html` や `styles.css` を編集して、表示内容やデザインを変更します。
3. 変更を公開する  
   GitHub に push すると、GitHub Pages 側に反映されます（反映まで少し時間がかかる場合があります）。

## ファイル構成（主要ファイル）

- `index.html`: 公開されるメインページ
- `styles.css`: ページのスタイル定義
- `.nojekyll`: Jekyll を無効化し、静的ファイルをそのまま配信する設定
